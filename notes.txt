TheSportsDB is free — no API key needed (uses public key "3").
Optionally override: add THESPORTSDB_API_KEY=<your_key> to .env for a Patreon key.
Add to .env: SYNC_SECRET=<random string>
Set up a cron job to POST /api/sync/matches — use cron-job.org (free) or Vercel Cron
Set cron to every 60s on match days, every 5 min otherwise
